# Brownie Support Site

This directory contains the static support pages for Brownie.

## Files

- `index.html`
  - Redirect entry point
- `en.html`
  - English support page
- `ja.html`
  - Japanese support page
- `styles.css`
  - Shared stylesheet

## Recommended Publishing Layout

Copy the contents of this directory to the root of a public static site repository.

Example:

```text
public-support-repo/
  index.html
  en.html
  ja.html
  styles.css
```

## Example URLs

- `https://<your-domain>/`
- `https://<your-domain>/en.html`
- `https://<your-domain>/ja.html`

## Microsoft Store

Recommended support URLs:
